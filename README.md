#  Multi-Tier(Student Manager App) Docker Compose Application

This project demonstrates a 3-tier architecture using Docker Compose with Node.js, MongoDB, and Nginx.

---

# Architecture

Client → Nginx → Node.js → MongoDB

---

# Tech Stack

- Node.js (Backend)
- MongoDB (Database)
- Nginx (Reverse Proxy)
- Docker & Docker Compose
- AWS EC2 (Deployment)
